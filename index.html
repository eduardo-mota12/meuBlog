scripts

Como os scripts funciona?

Os scripts sao codigos de programação que podem ser excutados e que fazem comandos continuos, contidos em um arquivo de script Tambem podem interagir com o sistema operacional banco de dados aplicativos jogos

scripts são sequências de comandos a serem executados, por isso podem ser perigosos. Algumas páginas na internet podem acionar scripts que burlam a segurança de navegadores, instalando vírus e outros malwares no seu computador.

por conta de malwares os navegadore bloqueiam qualquer tipo de scripts que nao seja autorizado pelo usuario excutar.
